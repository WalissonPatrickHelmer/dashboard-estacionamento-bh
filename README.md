# Dashboard de Estacionamento - Belo Horizonte

## 📊 Objetivo
Analisar dados de vagas de estacionamento rotativo em Belo Horizonte e gerar insights através de um dashboard no Excel.

## 📁 Dados
Os dados utilizados estão na pasta /data e representam:
- Bairro
- Tipo de estacionamento
- Total de vagas físicas
- Total de vagas rotativas
- Total de registros

## 🚀 Projeto em desenvolvimento
Dashboard será construído utilizando Excel 2016.
